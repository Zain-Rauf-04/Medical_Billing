# Medical_Billing
A modern medical billing platform for managing claims and payments.

## Features

- Patient management
- Medical billing
- Claims management
- Payment tracking
- Invoice generation
- Dashboard & reporting
- Secure and user-friendly interface

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/medi-bill-pro.git
```

2. Navigate to the project directory:

```bash
cd medi-bill-pro
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```
